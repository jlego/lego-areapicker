# com-areapicker 地区选择组件

### 参数说明：
- rootId: '117', 根节点ID
- fieldName: 'key', 表单域对应的键名
- width: 120, 下拉菜单宽度
- name: 'area',
- maxHeight: 300,
- nameArr: ['province', 'city', 'area'], //表单域名称 国家country, 省province, 市city, 区area
- placeholder: ['请选择省份', '请选择城市', '请选择区域'],
- value: [],
- selectOpts: {},
- onChange(self, result){}
